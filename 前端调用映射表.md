# CAPICS 前端调用映射表

## 1. 调用链说明

前端实际调用链分为 4 层：

1. `views/*.vue` 页面触发（`onMounted`、按钮点击、筛选变化）
2. `composables/*.js` 组合式封装（部分页面通过组合式间接调用）
3. `api/*.js` 请求封装（统一 `fetch('/api/...')`）
4. 后端接口（Nginx 反代到 Spring Boot）

## 4. 备注

- 所有 `api/*.js` 的基路径为 `API_BASE='/api'`，生产环境由 Nginx 反代到后端。
- 前端 token 存储键为 `localStorage.capics_token`，路由守卫在进入除 `/login` 外页面时检查该值。
- `ProductLineView.vue` 当前仅占位，尚未接入后端接口。
- `RealtimeView.vue` 为历史实时页，路由中未启用为主入口（当前主入口是 `/capacity-realtime` 与 `/capacity-realtime-monthly`）。
