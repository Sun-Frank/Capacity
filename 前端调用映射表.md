2. 页面到 API 映射
### 2.1 登录与公共
| 页面 | 前端函数 | 后端接口 |
| Login | `auth.login` | `POST /api/auth/login` |
| AppTopbar/退出 | `auth.logout` | `POST /api/auth/logout` |
| 初始化用户 | `auth.current` | `GET /api/auth/current` |
### 2.2 仪表盘（`/`）
| 动作 | 前端函数 | 后端接口 |
| 加载筛选项 | `mrp.getCreatedBys/getFileNamesByCreatedBy/getVersionsByCreatedByAndFileName` | `GET /api/mrp/filters/*` |
| 加载负载矩阵 | `dashboard.getLoadingMatrix` | `GET /api/dashboard/loading` |
| 动态模式快照列表 | `simulationSnapshot.getSnapshotNames` | `GET /api/simulation-snapshots/names` |
| 动态模式快照详情 | `simulationSnapshot.getSnapshot` | `GET /api/simulation-snapshots` |
### 2.3 MRP 对比（`/mrp-compare`）
| 动作 | 前端函数 | 后端接口 |
| 双文件筛选 | `mrp.getCreatedBys/getFileNamesByCreatedBy/getVersionsByCreatedByAndFileName` | `GET /api/mrp/filters/*` |
| 周对比数据 | `mrp.getWeeklyDemandSingle` | `GET /api/mrp/reports/weekly/single` |
| 月对比数据 | `mrp.getMonthlyDemandSingle` | `GET /api/mrp/reports/monthly/single` |
| 描述补齐 | `product.getProducts` | `GET /api/products` |
### 2.4 产品主数据（`/products`）
| 动作 | 前端函数 | 后端接口 |
| 编码族列表/搜索 | `product.getFamilies/searchFamilies` | `GET /api/products/families*` |
| 编码族导入/校验/模板 | `product.importFamilies/checkFamilyImportDuplicates/downloadFamilyTemplate` | `POST /api/products/families/import*`、`GET /api/products/families/template` |
| 定线列表/搜索 | `product.getFamilyLines/searchFamilyLines` | `GET /api/products/family-lines*` |
| 定线导入/校验/模板 | `product.importFamilyLines/checkFamilyLineImportDuplicates/downloadFamilyLineTemplate` | `POST /api/products/family-lines/import*`、`GET /api/products/family-lines/template` |
| 产品列表/搜索 | `product.getProducts/searchProducts` | `GET /api/products*` |
| 产品导入/校验/模板 | `product.importProducts/checkProductImportDuplicates/downloadProductTemplate` | `POST /api/products/import*`、`GET /api/products/template` |
### 2.5 MRP 计划（`/mrp`）
| 动作 | 前端函数 | 后端接口 |
| 三层筛选 | `useMrpFilters.*` | `GET /api/mrp/filters/*` |
| 计划列表 | `mrp.getPlansFiltered` | `GET /api/mrp/plans/filtered` |
| 周报/月报 | `mrp.getWeeklyReport/getMonthlyReportByFile` | `GET /api/mrp/reports/weekly/by-file`、`GET /api/mrp/reports/monthly/by-file` |
| 导入 MRP | `mrp.importMrpPlans` | `POST /api/mrp/plans/import` |
| 下载模板 | `mrp.downloadMrpTemplate` | `GET /api/mrp/plans/template` |
| 导出周报/月报 | `mrp.exportWeeklyReport/exportMonthlyReport` | `GET /api/mrp/reports/*/export` |
### 2.6 工艺路线（`/routing`）
| 动作 | 前端函数 | 后端接口 |

| 导入校验/导入/模板 | `routing.checkRoutingImportDuplicates/importRoutings/downloadRoutingTemplate` | `POST /api/routings/import/check`、`POST /api/routings/import`、`GET /api/routings/template` |
### 2.7 产线-产品（`/ct-line`）
| 动作 | 前端函数 | 后端接口 |

| 单条新增/编辑 | `ctLine.createCtLine/updateCtLine` | `POST /api/ct-lines`、`PUT /api/ct-lines/{id}` |
| 导入（同步） | `ctLine.importCtLines` | `POST /api/ct-lines/import` |
| 导入（异步） | `ctLine.importCtLinesAsync` | `POST /api/ct-lines/import-async` |

### 2.8 生产线配置（`/lines`）

### 2.9 静态产能（周/月）
| 页面 | 动作 | 前端函数 | 后端接口 |
|---|---|---|---|
| `/capacity-assessment` | 计算 | `capacity.getCapacityAssessment` | `GET /api/capacity-assessment` |
| `/capacity-assessment-monthly` | 计算 | `capacityMonthly.getCapacityAssessmentMonthly` | `GET /api/capacity-assessment/monthly` |
| 周/月页面 | 保存结果版本 | `simulationSnapshot.saveSnapshot` | `POST /api/simulation-snapshots` |
| 周/月页面 | 加载已保存版本 | `simulationSnapshot.getSnapshotNames/getSnapshot` | `GET /api/simulation-snapshots/names`、`GET /api/simulation-snapshots` |
> 说明：静态保存已改为“元数据提交 + 后端重算保存”，不再前端上传大 `linesData`。

| 周/月页面 | 快照保存 | `simulationSnapshot.saveSnapshot` | `POST /api/simulation-snapshots` |
| 周/月页面 | 快照读取 | `simulationSnapshot.getSnapshotNames/getSnapshot` | `GET /api/simulation-snapshots/names`、`GET /api/simulation-snapshots` |

| 动作 | 前端函数 | 后端接口 |

| 改密 | `user.changePassword` | `PUT /api/users/{id}/password` |
## 3. 备注
- API 基础路径固定为 `/api`，由 Nginx 统一反向代理到后端。
- 路由守卫 token 键：`localStorage.capics_token`。
- 前端主菜单结构以 `AppSidebar.vue` 为准。
