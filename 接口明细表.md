# CAPICS 接口明细表

## 1. 鉴权与返回约定
- 统一前缀：`/api`
- 鉴权方式：`Authorization: Bearer <token>`
- 放行接口：`/api/auth/**`、`/api/health`、`/api/lines/template`、`/api/ct-lines/template`
- 返回结构：`{ success, message, data }`

|---|---|---|---|
| POST | `/api/simulation-snapshots` | 是 | 保存快照 |

| GET | `/api/simulation-snapshots` | 是 | 快照详情 |

### 快照保存规则（重要）
- `source=dynamic`：可提交 `linesData/dates/dateLabels`，按前端结果保存。
- `source=static`：前端仅需提交元数据，后端会按 `createdBy/fileName/version/dimension` 重算后保存（用于规避大包体 413）。
