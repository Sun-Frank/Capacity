# MRP 对比页 Task

## 文档信息
- 更新日期：2026-04-25
- 文档版本：v1.2.2

## Task 1 路由与导航接入
- 输入：新页面组件路径与路由路径
- 输出：菜单和路由可进入 `MRP对比`
- 约束：沿用现有导航结构
- 验收：页面可从侧边栏访问

## Task 2 对比聚合逻辑
- 输入：文件A原始数据、文件B原始数据、产品主数据
- 输出：按描述与时间段汇总后的差异结果和统计卡片数据
- 约束：
  - 仅比较共同时间段
  - 描述来自产品主数据
  - 默认按差异绝对值倒序
- 验收：纯函数测试覆盖周/月、缺失描述、差异排序

## Task 3 页面筛选与加载状态
- 输入：文件A/文件B筛选条件、视图类型
- 输出：可触发查询并展示加载/空态/错误态
- 约束：两个文件都必须完整选择
- 验收：缺失条件时阻止查询

## Task 4 汇总卡片与差异表格
- 输入：聚合结果
- 输出：统计卡片、差异表格、颜色图例
- 约束：页面布局与现有系统一致
- 验收：行底色与差异方向一致

## Task 5 验证与收口
- 输入：实现代码
- 输出：测试通过、构建通过
- 验收：`node --test` 与 `npm run build` 均通过
